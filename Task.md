# 任務需求

1. 使用 Python 產生一個腳本，自動化將所有 git 專案中，所有分支都 pull 最新的變更。
   - 列出所有 本地分支。
   - 逐一 checkout 該分支。
   - 對該分支執行 git pull。
   - 遠端存在但本地尚未追蹤的分支，需先 git fetch --all 再決定是否建立追蹤分支
2. 腳本需要能在所有平台（Windows、macOS、Linux）上運行。
3. 腳本需要能夠處理多個 git 專案，並且能夠在每個專案中切換到所有分支並執行 pull 操作。
   - 腳本可掃描指定根目錄下的所有子目錄，尋找 git 專案。
   - 將每個子目錄視為一個 Git repository。
